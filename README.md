🚆 Railway Data Engineering Dashboard

📌 Project Overview

This project is a Streamlit-based Railway Data Engineering Dashboard that performs data exploration, transformation, analysis, and visualization on a railway dataset. It helps in understanding railway operations, performance patterns, and delays using data-driven insights.

The project is designed in a modular way where each stage of data processing is separated into different Python scripts for better structure and maintainability.


---

 Objectives

Perform data exploration on railway dataset

Clean and transform raw data

Analyze railway performance patterns

Visualize key insights interactively

Build a simple dashboard using Streamlit
