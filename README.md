# Railway Data Engineering and Analytics Dashboard

## Project Overview
This project is a Railway Data Engineering and Analytics Dashboard developed using Python, Pandas, Matplotlib, and Streamlit. The project performs data exploration, cleaning, transformation, aggregation, visualization, and reporting on railway operational data.

## Features
- Load and inspect railway dataset
- Perform data cleaning and preprocessing
- Filter trains based on operating days and stations
- Analyze source and destination station trends
- Generate visualizations for train operations
- Provide insights through an interactive Streamlit dashboard

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit
- Data Visualization
- Data Analysis
